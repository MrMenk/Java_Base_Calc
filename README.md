# 🧮 Java GUI Calculator Using Swing

A simple and user-friendly **desktop calculator application** built using **Java Swing** in NetBeans. This project demonstrates GUI development, event handling, and basic arithmetic operations in Java.

---

## 🚀 Features

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clean and intuitive UI
* Real-time input display
* Clear (`Clear`) and Backspace (`Back`) functionality
* Responsive button-based interaction

---

## 🛠️ Tech Stack

* **Language:** Java
* **GUI Framework:** Swing
* **IDE:** Apache NetBeans

---


## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Swing_Learn.git
   ```

2. Open in **NetBeans IDE**

3. Run the project:

   * Right-click project → **Run**

---

## 💡 Learning Outcomes

This project helped in understanding:

* Java Swing components (JFrame, JButton, JTextField)
* Event handling using ActionListener
* Layout management
* Basic application logic design
